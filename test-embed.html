Chat Widget Embed Test

Option 1: Eingebettetes Widget (iframe)

Das Widget wird direkt in die Seite eingebettet. Ideal fuer dedizierte Support-Seiten.

<iframe src="https://social-media-widget-ui.vercel.app" width="100%" height="600" style="border: none; border-radius: 12px;"></iframe>

Option 2: Floating Chat Button

Schau unten rechts - dort ist der Chat-Button. Klicke darauf um das Chat-Fenster zu oeffnen/schliessen.

Perfekt fuer: E-Commerce, Landing Pages, Blogs - ueberall wo der Chat nicht im Weg sein soll.

Siehe den Floating Button unten rechts auf dieser Seite!

Test: Scrollen

Scrolle auf dieser Seite um zu sehen, dass der Floating-Button immer sichtbar bleibt.
